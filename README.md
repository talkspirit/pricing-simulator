# Talkspirit Pricing Simulator

Simulateur de prix interactif pour les équipes Talkspirit.

## Accès

URL : `https://louissegers-byte.github.io/pricing-simulator/`

Accès protégé par mot de passe (demande à l'équipe).

## Stack

- HTML/JS vanilla, Tailwind via CDN, DM Sans
- Auth client-side PBKDF2 SHA-256 (1M itérations)
- Fichier unique self-contained — aucune dépendance, aucun build
- FR / EN / DE

## Usage

Ouvre l'URL, entre le mot de passe, simule, télécharge en PDF.
